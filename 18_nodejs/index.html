Node.js

Instalacion de Node

Instalar Node para el sistema operativo deseado. En el caso de Ubuntu irse a la pagina de node e ir a Descargas -> Instalando Node.js utilizando un gestor de paquetes -> Distribuciones de Linux basadas en Debian y Ubuntu, Enterprise Linux/Fedora y Snap -> GitHub 

    curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - &&\
    sudo apt-get install -y nodejs

Comprobar la instalacion con el comando node --version y deberiamos ver la version.

El poder de la terminal y como usar Node

Comandos basicos:

  • pwd: imprime el directorio actual
  • cd: cambiar de directorio
  • ls: lista el contenido del directorio actual
  • mkdir: crear carpeta/directorio
  • touch: crear archivo
Una vez instalado node, podemos escribir codigo javascript console.log("Hello World!"); y ejecutarlo por terminal con node index.js, deberiamos tener la salida:
Hello World!

REPL(Read Evaluation Print Loops) de Node

Es basicamente una terminal de JavaScript, igual a la que hemos estado usando en Google Chrome con los Developer Tools. Se ingresa escribiendo node en la terminal y estaremos dentro de REPL. Y podemos ejecutar comandos como console.log(), realizar operaciones, etc. Escribimos .exit para salir o presionar dos veces CTRL+C dos veces.

Como usar los modulos nativos de Node

Node viene instalado con muchos modulos, basicamente librerias que nos permiten hacer muchas cosas. Por ejemplo, para poder manipular el sistema de archivos local podemos usar el File System.

const

En lugar de declarar una variable con var <nombre> podemos declarar una constante con const <nombre>. La diferencia es la constante no se puede modificar. A diferencia de la variable no se puede modificar la variable mas adelante.

Ej: queremos copiar el contenido de un archivo en otro archivo. Esto se logra con el modulo fs (file system) y el metodo copyFileSync 

    index.js
    const fs = require("fs");
    fs.copyFileSync("file1.txt", "file2.txt");        
            terminal
            node index.js

Lo primero es declarar una constante y requerir el modulo. Luego llamamos al metodo de la constante y copiamos el contenido de file1.txt en file2.txt. En caso el archivo ya exista simplemente se reemplaza. Al ejecutar el codigo se crear√° file2.txt con el contenido de file1.txt

npm e instalar modulos externos de node

npm viene instalado por defecto cuando instalamos node. Podemos setear las opciones iniciales del proyecto corriendo npm init y siguiendo paso a paso; al terminar vemos que se ha creado un package.json que es basicamente la descripcion de nuestro proyecto. Podemos buscar modulos para instalar desde npmjs.com.
Como ejemplo instalaremos el paquete superheroes con: 

    npm install superheroes

Y modificaremos el archivo index.js para que requiera el paquete e imprima un nombre aleatorio de superheroe: 

    const superheroes = require('superheroes'); 
    console.log(superheroes.random());

Luego simplemente lo corremos con node index.js. Podemos observar tambien que se crea el directorio node_modules que contiene todos los paquetes instalados.
