Calculadora de BMI con ifs

    function bmiCalculator (weight, height) {
        var BMI = Math.round(weight/(height*height));
        if (BMI < 18.5) {
            return ("Your BMI is "+BMI+", so you are underweight.");
        }
        if (BMI >=18.5 && BMI <= 24.9) {
            return ("Your BMI is "+BMI+", so you have a normal weight.");
        }
        if (BMI > 24.9) {
            return ("Your BMI is "+BMI+", so you are overweight.");
        }
    }

Arrays

  • Lista, coleccion de items. Puede contener elementos de diferentes tipos.
  • Declarar: var <array> = [<item>, ..., <item>];
  • Se puede imprimir con console.log(<array>)
  • Acceder al elemento: <array>[<indice>]
  • Acceder al ultimo elemento: <array>[<tama침o-1>]
  • Tama침o: <array>.length
  • Si un elemento existe: <array>.includes(<elemento>)
  • Agregar elementos al final del array: <array>.push(<elemento>)
  • Eliminar el ultimo elemento del array: <array>.pop()
Ejemplo: 
    var mollendo = ["Anibal", "Christian", "Gino", "Martin"];
    var nombre = prompt("Nombre: ");
    console.log(mollendo.includes(nombre));

FizzBuzz: agregar a un array elementos, si el elemento es multiplo de 3 poner Fizz, si es multiplo de 5 poner Buzz, si es multiplo de ambos poner FizzBuzz.

    var output = [];
    var count = 1;
    
    function FizzBuzz(){
        if(count%3===0 && count%5===0){
            output.push("FizzBuzz");
        }else if(count%3===0){
            output.push("Fizz");
        }else if(count%5===0){
            output.push("Buzz");
        }else{
            output.push(count);
        }
        count++;
        console.log(output);
    }
    
    FizzBuzz();
