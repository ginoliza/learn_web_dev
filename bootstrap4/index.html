Instalar bootstrap

Para instalar bootstrap podemos ir a getbootstrap.com y descargar el codigo fuente de bootsrap

La manera mas simple para usar Bootstrap es poner el CDN del CSS dentro del tag link en el head para los estilos

La otra forma es simplemente copiar el codigo de esta pagina. Lo unico que nos interesa es el head basicamente.

La tercera forma es incluirlo localmente al descargar Bootstrap. Pero esto niega cualquier opcion de caching.
