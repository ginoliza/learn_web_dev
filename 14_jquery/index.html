JQuery

Es una libreria (como Bootstrap) de JavaScript. Muchas veces escribir javascript es muy complicado y dificil de leer, jquery simplifica esto.
Por ejemplo, ambos codigos son equivalentes: 

    javascript
    document.querySelector("h1");                
    jquery
    jQuery("h1");
    // o mas simple
    $("h1");

Seleccionar elementos con jquery

El signo $("drum") en jquery basicamente reemplaza al document.querySelectorAll("drum")

Manipulando estilos con jquery

Podemos usar el metodo .css("<propiedad>", "<valor>") para establecer valores de CSS, o simplemente .css("<propiedad>") para poder acceder al valor de esa propiedad. Pero hemos aprendido que no esta bien manipular estilos dentro de javascript que deberia ser solo comportamiento. 

    -- acceder --
    $("h1").css("color");

    > 'rgb(255, 255, 0)'                
    -- establecer --
    $("h1").css("color", "red");

Entonces es mejor declarar alguna clase dentro del .css y agregarla dentro del .js con el metodo .addClass("<valor>"). 

    -- index.css --
    .big-title{
        font-size: 10rem;
        color: yellow;
        font-family: cursive;
    }                
    -- index.js --
    $("h1").addClass("big-title");

Para agregar multiples clases se ponen dentro de las comillas (" ") y separando con espacios: 

    $("h1").addClass("clase1 clase2");

Para quitar la clase se usa el metodo .removeClass(<clase>) 

    $("h1").removeClass("big-title");

El metodo .hasClass(<clase>) se usa para verificar si un elemento tiene una clase o no. Esto retorna un valor booleano. 

    $("h1").hasClass("big-title");
    > true

Manipular texto

La primera forma es usar el metodo .text("<Texto>") (equivalente a .textContent).
Ej: 

    $("h1").text("Otra cosa");

Otra forma es usar el atributo .html("<texto>") (equivalente a innerHTML) y reemplazar literalmente todo el HTML dentro de ese elemento.
Ej: 

    $("h1").html("<em>Algo cursivo<em>");

Manipular atributos

Con el metodo .attr("<atributo>") podemos acceder y establecer atributos como src href e incluso class, etc
Ej: 

    -- acceder --
    $("h1").attr("class");
    > 'big-title'                
    -- establecer --
    $("h1").attr("class", "huge-title");

Event Listeners

Clicks

Podemos simplemente usar .click(<funcion>) en lugar de su equivalente en javascript document.querySelector("<selector>").addEventListener("click", <funcion>);
Ej: 

    $("h1").click(function () {
        $("h1").css("color", "purple");
    });

Si queremos cambiar el color de TODOS los botones en jquery es mas facil, ya que no tenemos que iterar, obtener el tama침o del array, etc.
Ej: 

    -- jquery --                    
    $("button").click(
        function () {
            $("button").css("background-color", "purple");
        }
    );
 
    -- javascript --                    
    document.querySelectorAll("button").addEventListener("click",function () {
        document.querySelectorAll("button").style.backgroundColor = "purple";
        }
    );

Teclado

Podemos usar el metodo .keypress("<funcion>") y el event para un elemento especifico:

    $("input").keypress(function (event) {
        console.log(event.key);
        });

O tambien podemos detectar teclas presionadas en toda la pagina web sin especificar un selector, sino todo el documento: 

    $(document).keypress(function (event) {
        console.log(event.key);
        });

RETO

Que la tecla presionada en la pagina web se muestre en el h1. 

    $(document).keypress(function (event) {
        $("h1").text(event.key);
    });

Metodo .on

Algo mas flexible puede ser usar el metodo .on("evento", <funcion>);. El primer parametro es el evento que se esta escuchando y el segundo parametro es lo que pasa cuando el evento ocurre. Lista de eventos
Ej: 

    $("h1").on("mouseover", function () {
        $("h1").css("color", "purple");
    });

Agregar y quitar elementos

Con los metodos .before("<html>") .after("<html>") .prepend("<html>") .append("<html>").

  • before: agrega el elemento antes del elemento seleccionado
  • after: agrega el elemento despues del elemento seleccionado
  • prepend: agrega el elemento antes del contenido del elemento seleccionado
  • arepend: agrega el elemento despues del contenido del elemento seleccionado
    • Ej: 
    $("h1").after("<h1> Hello2 </h1>");

Y para eliminar elementos usamos el metodo .remove();
Ej: 

    $("button").remove();

Animaciones

Podemos usar los metodos: .hide() .show() o alternar entre ambos con .toggle().
Puede que esto parezca muy brusco, asi que hay animaciones como .fadeOut() .fadeIn() .fadeToggle() .slideUp() .slideDown() .slideToggle()
Tambien podemos usar nuestra propia animacion con el metodo .animate({<codigo css>}): 

    $("h1").animate({opacity: 0.5});

Pero aqui podemos solamente animar con valores numericos, por ejemplo no podemos cambiar a color: red; ya que no es un valor numerico.

Concatenar animaciones

Las animaciones se pueden concatenar para que ocurran una despues de la otra: 

    $("h1").slideUp().slideDown().animate({opacity: 0.3});
