JQuery

Es una libreria (como Bootstrap) de JavaScript. Muchas veces escribir javascript es muy complicado y dificil de leer, jquery simplifica esto.
Por ejemplo, ambos codigos son equivalentes: 

    javascript
    document.querySelector("h1");                
    jquery
    jQuery("h1");
    // o mas simple
    $("h1");

Incorporar jquery

Al igual que bootsrap podemos descargarlo y usarlo, o simplemente podemos usar un CDN, la opcion mas popular de CDN es usar el de google. Vamos a usar la version 3.x. La posicion es muy importante, generalmente es buena practica ponerlo arriba de nuestro propio .js, ya que HTML se lee de arriba a abajo y vamos a usar jquery dentro de nuestro .js
Otra cosa de la que debemos tener cuidado es incluirlo en el head, porque incluso si posicionamos jquery encima del propio .js puede no funcionar pero tampoco dar errores. Podemos poner una restriccion a esto preguntando si el documento ya está listo (que las librerias hayan terminado de cargar) y recien ejecutar el codigo: 

    -- index.js --
    $(document).ready(function(){
        $("h1").css("color", "red");
    });

Pero en el futuro simplemente pondremos las librerias antes de cerrar el body

Minificación para reducir tamaño

Si entramos al link dentro del atributo src de el elemento script podemos ver la fuente tanto de bootsrap, como de jquery. Y darnos cuenta que es bastante codigo pero sin espacios, indentaciones, comentarios, etc. Esto con el fin de ahorrar espacio. Podemos hacerlo nosotros tambien con alguna herramienta online: Minifier

Seleccionar elementos con jquery

El signo $("drum") en jquery basicamente reemplaza al document.querySelectorAll("drum")

Manipulando estilos con jquery

Podemos usar el metodo .css("<propiedad>", "<valor>") para establecer valores de CSS, o simplemente .css("<propiedad>") para poder acceder al valor de esa propiedad. Pero hemos aprendido que no esta bien manipular estilos dentro de javascript que deberia ser solo comportamiento. 

    -- acceder --
    $("h1").css("color");

    > 'rgb(255, 255, 0)'                
    -- establecer --
    $("h1").css("color", "red");

Entonces es mejor declarar alguna clase dentro del .css y agregarla dentro del .js con el metodo .addClass("<valor>"). 

    -- index.css --
    .big-title{
        font-size: 10rem;
        color: yellow;
        font-family: cursive;
    }                
    -- index.js --
    $("h1").addClass("big-title");

Para agregar multiples clases se ponen dentro de las comillas (" ") y separando con espacios: 

    $("h1").addClass("clase1 clase2");

Para quitar la clase se usa el metodo .removeClass(<clase>) 

    $("h1").removeClass("big-title");

El metodo .hasClass(<clase>) se usa para verificar si un elemento tiene una clase o no. Esto retorna un valor booleano. 

    $("h1").hasClass("big-title");
    > true
