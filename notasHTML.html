Hay elementos HTML y cada elemento tiene atributos, algunos elementos se cierran, otros no.
Todo esto se puede ver en la documentacion, existen multiples sitios de documentacion, uno de ellos es https://devdocs.io
Para poner el boilerplate de HTML en VSCODE se hace con !
Un compilador online para ver HTML, CSS, Javascript es https://codepen.io
https://docs.emmet.io es una buena pagina para ver todos los emmets
Los HTML se pueden abrir simplemente en el navegador
Iconos:

  • En flaticon.com hay buenos iconos gratis.
  • https://www.favicon.cc/ es un buen recurso para iconos. Podemos dibujarlos. Se incluyen en el head con un link igual que un .css. Pero su <...rel="icon"...>
pesticide es una buena herramienta que muestra las cajitas del HTML. Mantener presionado ctrl y mover el mouse sobre algun elemento nos muestra su descripcion.
css3buttongenerator.com es un generador de botones para la forma, color, hover, etc.

Secciones HTML

head

Contiene informacion de la pagina web y le dice al navegador como manejar la pagina
title es el titulo de la pestaña
meta(elemento) charset (atributo) es la codificacion del site
utf-8 es la codificacion estandar
https://unicode-table.com es una buena pagina de referencia
description (atributo) es lo que sale cuando se hace una busqueda en google

body

Todo el cuerpo de la pagina

Elementos HTML

Cabeceras

h1, h2, h3, h4, h5, h6 son las cabeceras

Interlineado

Para usar saltos de linea se usa br

Para lineas horizontales se usa hr

Estilo de texto

p es para poner texto, parrafo

strong o b es para poner negrita
siempre es mejor poner strong

em o i es para poner énfasis o italica
siempre es mejor usar em

Listas

ul es una lista no-ordenada. Por defecto tiene puntos. Puede tener elementos:
    ol lista ordenada. Por defecto tiene numeros
    li se usa para listar los elementos de la lista

Imagenes

img para poner imagenes, no necesita cerrarse.
    src(atr) es la fuente de la imagen, puede ser una URL o imagen local
    alt(atr) es un texto que se muestra cuando no pueda cargar la imagen
    width, height(atr) para el ancho y altura respectivamente

Links

a se usa para poner hiperlinks
    dentro de href(atr) va el link

Tablas

Primero filas, luego columnas
table
    tr table row, filas
    td table data, elementos de la fila

Pero si queremos ser mas especificos podemos usar
thead para la cabecera y th en vez de td
tbody para el cuerpo
tfoot para el pie de pagina
Normalmente se usan los dos primeros thead y tbody
Se pueden anidar tablas

Formularios

  • form tiene adentro:
    • action(atr) es la pagina a donde lleva el form
      • mailto:email@mail.com
    • label va junto al campo a llenar
    • input:
      • type(atr) es el tipo del input, puede ser:
        • text cualquier texto
        • submit para el boton
        • color para seleccionador de color
        • checkbox para un checkbox
        • password para un campo password
        • email para un email, valida @, etc
        • etc, etc
    • textarea:
      • name(atr)
      • rows(atr): filas o largo
      • cols(atr): columnas o ancho
